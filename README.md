# 🌍 Topografie Quiz — Spelletje

Een klein topografiespel: raad welke **hoofdstad** bij welk **Europees land** hoort.

- 15 willekeurige vragen per ronde
- 4 antwoordopties per vraag
- Bij een fout antwoord zie je meteen de juiste hoofdstad
- 44 Europese landen

## Spelen

Open `index.html` in je browser, of speel de gepubliceerde versie via GitHub Pages.

## Ontwikkeling

De spellogica zit in `quiz.js` en de dataset in `quiz-data.js`. Beide werken
zowel in de browser als in Node, zodat de logica getest kan worden:

```
node quiz.test.js
```
